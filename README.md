# To-Do List App

A simple and interactive To-Do List web application built using HTML, CSS, and JavaScript.  
This project allows users to add tasks, manage them, and delete completed tasks.

## Features
- Add new tasks
- Delete tasks
- Simple and clean user interface
- Responsive design

## Technologies Used
- HTML5
- CSS3
- JavaScript (DOM Manipulation)
